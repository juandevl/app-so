Aplicación

El proyecto consta de una aplicación web(CRUD), que permite insertar juegos de mesa, actualizarlos, eliminarlos y mostrar todos los datos almacenados.

Esta aplicación web está respaldada por una infraestructura basada en contenedores Docker.
El sistema consta de dos contenedores: uno personalizado que combina Apache HTTP Server, WSGI, Python y JavaScript para manejar la lógica de la aplicación web y las interacciones del usuario, y otro que utiliza MongoDB como base de datos para el almacenamiento y gestión de datos.
